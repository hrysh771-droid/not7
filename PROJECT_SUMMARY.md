# ملخص مشروع منصة قراءة الروايات الإلكترونية

## نظرة عامة

## 🔧 API Endpoints

### المصادقة
- `POST /api/auth/register` - إنشاء حساب
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/auth/me` - بيانات المستخدم
- `PUT /api/auth/profile` - تحديث الملف الشخصي

### الروايات
- `GET /api/novels` - قائمة الروايات
- `GET /api/novels/:id` - تفاصيل رواية
- `GET /api/novels/search` - البحث
- `GET /api/novels/popular` - الروايات الشائعة
- `POST /api/novels/:id/favorite` - إضافة للمفضلة

### الفصول
- `GET /api/chapters/novel/:novelId` - فصول رواية
- `GET /api/chapters/:id` - تفاصيل فصل
- `GET /api/chapters/:id/content` - محتوى الفصل
- `POST /api/chapters/:id/progress` - تحديث التقدم

### الإدارة
- `GET /api/admin/dashboard` - إحصائيات
- `GET /api/admin/users` - إدارة المستخدمين
- `GET /api/admin/novels` - إدارة الروايات
- `GET /api/admin/comments` - إدارة التعليقات
